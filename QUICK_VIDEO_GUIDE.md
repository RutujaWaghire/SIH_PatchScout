## 🔧 IF SOMETHING GOES WRONG

### Site Not Loading?
```powershell
# Restart both servers:
.\START_SERVERS.ps1
```

### Scan Not Starting?
- Check console (F12) for errors
- Refresh page (Ctrl + R)
- Try again with only Nmap selected

### Page Looks Broken?
- Clear browser cache
- Hard refresh (Ctrl + Shift + R)
- Close and reopen browser
